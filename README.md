To get the script
```
curl -o 0g-storage-node-setup.sh https://raw.githubusercontent.com/jackleeinitianode/0g-storeage-node/main/0g-storage-node-setup.sh &&  \
chmod +x 0g-storage-node-setup.sh &&  \
./0g-storage-node-setup.sh
```
